Git

  • git status - показывает состояние рабочего каталога и индекса. Это позволяет проверить, какие изменения были подготовлены, какие ещё не были подготовлены и какие файлы не отслеживаются Git.
  • git checkout - используется для переключения на указанную ветку в репозитории
  • pwd - расшифровывается как "Present working directory", что переводится как "текущая рабочая директория". То есть команда pwd показывает, в какой сейчас папке мы находимся.
  • git branch -используется для создания новой ветки с указанным именем, которая будет ответвлением от текущей ветки.
  • git mv - используется для безопасного перемещения или переименования файлов и каталогов в репозитории./
